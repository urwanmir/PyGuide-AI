# PyGuide AI

PyGuide AI is a minimalist, AI-powered Python learning assistant designed for everyone, from absolute beginners to experts. It features a clean, ChatGPT-like interface and provides patient, analogies-driven tutoring to make learning Python fun and accessible.

Made with ❤️ by **Hadi (Urwan Mir)**.

## 🚀 Features

- **AI Python Tutor**: Friendly and patient AI powered by Gemini 3 Flash.
- **Minimalist UI**: Clean, distraction-free interface inspired by ChatGPT.
- **Data Portability**: Export and import your chat history as JSON files.
- **Open Source**: Built for the community.
- **Beginner Friendly**: Designed to be usable by anyone, including non-technical users.

## 🛠️ Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS, Motion (Framer Motion).
- **AI**: Google Gemini API (@google/genai).
- **Icons**: Lucide React.
- **Deployment**: Optimized for Vercel.
